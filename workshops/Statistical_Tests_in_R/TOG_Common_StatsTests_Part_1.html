Hypothesis testing

  • Scientific hypothesis can be translated into statistical hypothesis/statements

  • Statistical hypothesis testing provides estimates for the magnitude and certainty we have for a scientific hypothesis

  • Key is usually is to ask how consistent our data is with a specified hypothesis

Hypothesis testing

  • Usually we frame our statistical hypotheses in terms of a ‘null’ and ‘alternative’ hypothesis

  • Null: there is no effect/change in what we are measuring

  • Alternative: there is some change (increase, decrease, etc) in whatever we are measuring

  • Sometimes denoted \(H_0\) and \(H_A\)

Hypothesis testing

  1. Generate hypothesis
  2. Collect data
  3. Test hypothesis
  4. Interpret results

Sometimes 1 and 2 are flipped

Key is usually in 4 - must be combination of background and statistical output

Types of errors

  1. Type 1 error: probability of rejecting the null hypothesis when the null hypothesis is actually true.

  2. Type 2 error: probability of failing to rejecting the null hypothesis when it is false

Note: statistical power is 1-Type 2 error, i.e. probability of correctly identifying a true effect.

p-values

  • Most commonly used way of hypothesis testing

  • Controversial, often overused, misinterpreted, but pervasive

  • Informally, the p-value is the probability under a specified statistical model that a statistical summary of the data would be equal to or more extreme than its observed value

  • Often use a cutoff of 0.05 or 5%

p-values

DOES:

  • Indicate how incompatible data are with a prespecified statistical model. E.g., model in which two groups have the same outcome

DOES NOT:

  • Give the probability of the null hypothesis being true!

  • Measure the size of the effect/association!

  • Exist in a vacuum!

‘Whenever a researcher chooses what to present based on statistical results, valid interpretation of those results is severely compromised if the reader is not informed of the choice and its basis’

Confidence interval

  • Hypothesis tests are meant to look for a statistical difference

  • These differences have uncertainty

  • Confidence interval shows range of differences data is compatible with

  • Small p-value = more narrow confidence interval

Hypothesis testing: Presenting results

  1. Summary data

  2. Difference (absolute or relative or both)

  3. Confidence interval for difference

  4. p-value

Types of tests

  • Depends on the outcome and exposure type (continuous, binary etc)

  • Parametric vs nonparametric

  • In general most ‘simple’ tests can be extended to or are very similar to regression models

Continuous outcomes

When outcome is continuous we have a few options:

  1. T-tests (binary exposure)

  2. ANOVA (categorical exposure)

  3. Correlation coefficients (continuous exposure)

  4. Linear regression models (any exposure)

T-tests

To run a t-test in R, we can use t.test() function, which takes two vectors (group 1 and group 2) and a command ‘paired’

group1 <- sleep %>% filter(group == 1)
group2 <- sleep %>% filter(group == 2)

t.test(group1$extra, group2$extra, paired = TRUE)
## 
##  Paired t-test
## 
## data:  group1$extra and group2$extra
## t = -4.0621, df = 9, p-value = 0.002833
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -2.4598858 -0.7001142
## sample estimates:
## mean of the differences 
##                   -1.58

Aside: broom package

As we can see, the output from t.test() is a bit messy to work with. Luckily R has the wonderful ‘broom’ package, which works with all sorts of tests/regression models to ‘clean-up’ output.

Non parametric alternatives

When one is unsure about the distribution of data (e.g. very skewed, or ordinal scale), a nonparametric alternative can be useful as main test (or sensitivity analysis)

  1. For two-sample t-test -> Mann-Whitney test (`wilcox.test(..., paired = FALSE))

  2. For paired t-test -> Wilcoxon signed rank test (`wilcox.test(..., paired = TRUE))

  3. For ANVOVA ->Kruskal Wallis (`kruskal.test(...))

Binary

  • When the outcome is binary - we no longer summarize things with means or mean differences.

  • Usually one uses relative measures (e.g. risk or odds ratios)

  • Tests for association are different than above but presentation should be similar!

Two binary variables

Have two main options for testing relationship between two binary variables

  1. Fisher exact test (good for small sample sizes, e.g. any cell size < 5)

  2. \(\chi^2\) test (good most of the time)

Summary

  • When testing we should usually present effect estimates and 95% CIs as well as p-values

  • Type of test depends on outcome/exposure variable type, but workflow is similar in R

  • Generally speaking, regression models can be viewed as extensions of simple tests, and can always be used as a substitute
